# Changelog

## 0.2.0 (2025-06-10)

### Added
- New query filter methods: `filter_not_in()`, `filter_like()`, `filter_ilike()`, `filter_is_null()`, `filter_is_not_null()`, `filter_regex()`, `filter_between()`, and `filter_or()`
- Enhanced `_build_query()` method to support all new operators and improved parameter handling
